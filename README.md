# VPN OnOff

Android app that automatically toggles Clash Meta VPN based on WiFi connectivity.

- WiFi disconnected → Start VPN
- WiFi connected → Stop VPN

## Requirements

- Android 8.0+
- [Clash Meta for Android](https://github.com/MetaCubeX/ClashMetaForAndroid) installed and configured

## Permissions

- **MIUI/HyperOS devices**: Grant "后台弹出界面" (Background popup) permission for background operation
- **Overlay permission**: Required for background activity control
